1. Pour visualiser correctement les ecrans a l'execution, veuillez à changer la variable plotX dans la classe CVM :

- Si votre ecran est un 13 pouces alors il faut mettre 340.
- Si votre ecran est au dessus de 15 pouces alors il faut mettre 480.

2. Pour lancer le scenario d'integration en mode single jvm :
- lancer la classe CVM

3. Pour lancer le scenario d'integration en mode multi jvm :
- lancer le script start-gregisrty
- lancer le script start-cyclicbarrier
- lancer le script start-dcvm
